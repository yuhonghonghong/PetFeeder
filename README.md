# 一、智能宠物喂食器
##  一款自动控制和远程控制的宠物投食机
# 二、方案应用场景及功能概述
##  家庭-客厅
##  功能概述
* 投食方式
  * 自动投食。通过设备上的按键一键投食,同时上传投食记录便于用户查看
  * 远程投食。通过app上的投食功能按键远程投食
* 创建喂食计划
  * 用户通过app创建一个投食计划。投食计划包括时间间隔，喂食份数等参数设置。需要同步到本地设备端。投食方式为自动投食或到点提示手动投食方式
* 小夜灯
* 设备端电池电量检测
